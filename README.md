# Business Address Book

A modern React application for organizing business contacts by geographic location.

## Features

- Geographic Organization by State, City, and Locality
- Contact Management with phone, email, and notes
- Search Functionality
- Edit & Delete operations
- Responsive Design
- Modern UI with Tailwind CSS

## Installation

```bash
git clone https://github.com/YOUR-USERNAME/business-address-book.git
cd business-address-book
npm install
npm start
